Create snake game using pygame library 

for installing pygame use
``` !pip install pygame ```

Functionality of the game are:
* Snake length increases on eating the fruit
* Score changes eating the fruit
* cheats can be used to either increase the size of the snake or the score
* hi score is updated when you beat it
* sound effects
* snake dies when it hits the border
  
controls for the game are:- 
* Use the arrow keys for navigating the Snake
* Q and E can be used as cheats as they increase the score and length of the snake but the score made using cheats will not be saved to hi-score

the hi-score is stored in the hi-score.txt file
